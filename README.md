# Python Utility Suite

A collection of beginner-friendly Python projects developed to strengthen programming fundamentals and problem-solving skills. This repository includes practical command-line applications built using core Python concepts such as functions, loops, file handling, conditional statements, and user input validation.

## Projects Included

### 1. To-Do List Application

A command-line task management application.

#### Features
- Add new tasks
- View all tasks
- Delete tasks
- Persistent task storage using text files
- Input validation

#### Concepts Used
- File Handling
- Lists
- Loops
- Functions
- Exception Handling

A simple calculator that performs basic arithmetic operations.

#### Features
- Addition
- Subtraction

- Division

#### Concepts Used
- Functions

A command-line password generator that creates secure passwords based on user requirements.

#### Concepts Used
